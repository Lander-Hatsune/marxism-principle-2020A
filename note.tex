\item 东方社会也要经历西方的道路
\end{itemize}
\item 1871 - 1883 晚年
\begin{itemize}
\item \textbf{1871: 法兰西内战和巴黎公社}
\item 东方社会具备意志性, 有可能不经过资本主义的卡夫丁峡谷, 直接进入社会主义
\end{itemize}
